# お題ストック

毎日の英語日記のためのお題集。気分とフェーズに合わせて自由に選んでください。

## カテゴリ一覧

| ファイル | テーマ | おすすめのフェーズ |
| --- | --- | --- |
| [`daily-life.md`](./daily-life.md) | 日常の小さな出来事 | Phase 1 / 2 / 3 |
| [`emotions.md`](./emotions.md) | 感情・気分の動き | Phase 1 / 2 / 3 |
| [`observation.md`](./observation.md) | 観察・気づき | Phase 1 / 2 / 3 |
| [`design-and-work.md`](./design-and-work.md) | デザイン・仕事 | Phase 2 / 3 |
| [`reading-and-writing.md`](./reading-and-writing.md) | 読書・短歌・エッセイ | Phase 2 / 3 |
| [`ielts-style.md`](./ielts-style.md) | IELTS 的なお題 | Phase 3 後半〜 |

## 選び方のヒント

- **疲れてる日** → `daily-life` の食べ物や天気の話
- **頭が動く日** → `observation` か `design-and-work`
- **心が動いた日** → `emotions`
- **本を読んだ後** → `reading-and-writing`
- **試験対策モード** → `ielts-style`

## ストックが減ってきたら

`prompts/topic-generator.md` を使って AI に追記してもらえます。

```
@prompts/00-profile.md と @prompts/topic-generator.md を読んで。
@topics/daily-life.md に新しいお題を 15 個追記して。
既存の番号と被らないように続き番号で。
```

## 使ったお題に印をつける運用（任意）

- `~~取り消し線~~` で「使った」を表す
- 行末に ` — done YYYY-MM-DD` を付け足す
- もしくは何もしない（また書きたくなるかもしれないので）
